# Espruino-WebSockets
Basic Espruino program to connect an Espressif chip to a WebSockets server and send/receive data

## Flash package
NodeJoy CDN hosts a complete flashing package with blank binaries to overwrite previously programmed Espressif devices @ [https://NodeJoy.com/CDN/Espruino/Bin/Flash.zip](https://nodejoy.com/cdn/espruino/bin/flash.zip)
